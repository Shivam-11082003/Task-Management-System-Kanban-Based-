# Task Management System (Kanban Based)

## Tech Stack
Backend: Node.js, Express, MongoDB
Frontend: React + Vite

## Backend Setup
cd backend  
npm install  
Create .env file using .env.example  
npm start  

## Frontend Setup
cd frontend  
npm install  
npm run dev  

## Features
- User Authentication (Register/Login/Logout)
- Profile Update & Delete
- Task CRUD
- User Specific Tasks
- Filter by status
- Kanban Board with Drag & Drop
